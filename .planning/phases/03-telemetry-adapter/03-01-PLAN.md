| telemetry client → Prometheus | HTTPS/HTTP to configured baseUrl only; no user-supplied query URLs |
| npm registry → monorepo | External packages installed via pnpm |

| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-03-02 | Tampering | PromQL via flagKey/variationId | mitigate | Label escaping deferred to plan 02 build-promql.ts; D-04 rejects raw user PromQL |
| T-03-03 | Spoofing/Tampering | SSRF via baseUrl | mitigate | Validate baseUrl is http(s); client only calls configured baseUrl + /api/v1/query |
